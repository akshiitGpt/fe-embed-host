Embedding a Remote Widget into a Host App

This host app demonstrates two ways to embed a frontend widget served from GitHub Pages — without including any of the widget's source code.

1 iFrame Embed

The widget is loaded directly inside an iframe. The host app has zero knowledge of the widget's internals.

<iframe
  src="https://akshiitgpt.github.io/fe-embed-widget/"
  width="380"
  height="420"
  style="border:none;border-radius:12px"
></iframe>

2 Script Tag Embed

A single script tag injects a floating chat button. Click the chat bubble in the bottom-right corner to try it.

<script
  src="https://akshiitgpt.github.io/fe-embed-widget/embed.js"
  data-position="bottom-right"
></script>

^ The floating chat icon in the bottom-right corner is injected by this script.

3 Cross-Origin Communication
